⚡ GitHub Pages in 3 steps

deployment-guide 
1️⃣ Create a new repository on GitHub → named: your-username.github.io

2️⃣ Upload or commit this index.html file to the main branch

3️⃣ Go to Settings → Pages → branch: main → / (root) → Save

🎉 Your site will be live at https://your-username.github.io

✨ Pro tip: You can also use a custom domain or different project repo.
